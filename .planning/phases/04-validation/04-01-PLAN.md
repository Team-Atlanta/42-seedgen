Script structure:
- Shebang: #!/bin/bash
- Accept one positional argument: BASELINE_DIR (output directory)
- Error handling: set -e, set -o pipefail
- Configuration:
  - HARNESS_PATH="${HARNESS_PATH:-./artifacts/coverage-harness}" (directory containing harness binary)
  - Find harness binary: HARNESS=$(find "$HARNESS_PATH" -type f -executable | head -1)
  - Verify harness exists: test -n "$HARNESS" || exit 1
- Create output directory: mkdir -p "$BASELINE_DIR"
- Run harness with empty input to establish baseline:
  - echo "" | "$HARNESS" > /dev/null 2>&1 || true  # Allow non-zero exit
- Merge raw profiles:
  - llvm-profdata merge -sparse default.profraw -o "$BASELINE_DIR/baseline.profdata"
- Generate JSON coverage report:
  - llvm-cov export "$HARNESS" -instr-profile="$BASELINE_DIR/baseline.profdata" -format=text > "$BASELINE_DIR/coverage.json"
- Extract and echo metrics using Python one-liner:
  - python3 -c "import json; data=json.load(open('$BASELINE_DIR/coverage.json')); totals=data['data'][0]['totals']; print(f\"Baseline: {totals['branches']['covered']}/{totals['branches']['count']} branches\"); print(f\"Baseline: {totals['functions']['covered']}/{totals['functions']['count']} functions\")"
- Make executable: chmod +x

Follow Pattern 2 from 04-RESEARCH.md (Baseline Coverage Measurement).
Handle profraw cleanup between measurements.

  <action>
Create Python script that compares baseline vs seed coverage and asserts improvement.

Script structure:
- Shebang: #!/usr/bin/env python3
- Imports: import json, sys, argparse
- Function load_coverage(path: str) -> dict:
  - Opens JSON file and loads data
  - Extracts totals = data['data'][0]['totals']
  - Returns dict with keys: branches_covered, branches_total, functions_covered, functions_total
- Function main():
  - Parse two positional arguments: baseline_json, seeds_json
  - Load both coverage files using load_coverage()
  - Calculate improvements:
    - branch_improvement = seeds['branches_covered'] - baseline['branches_covered']
    - function_improvement = seeds['functions_covered'] - baseline['functions_covered']
  - Print comparison:
    - Baseline: {branches_covered}/{branches_total} branches
    - With seeds: {branches_covered}/{branches_total} branches
    - Improvement: +{branch_improvement} branches ({percentage}%)
    - (same for functions)
  - Assert measurable improvement:
    - assert branch_improvement > 0 or function_improvement > 0, "VALIDATION FAILED: Seeds did not improve coverage over baseline"
  - Print: "✓ VALIDATION PASSED: Seeds demonstrate measurable coverage improvement"
  - sys.exit(0)
- if __name__ == "__main__": main()
- Make executable: chmod +x

Follow Pattern 4 from 04-RESEARCH.md (Coverage Comparison and Assertion).
Use relative comparison (improvement >0%) not absolute thresholds per research guidance.
