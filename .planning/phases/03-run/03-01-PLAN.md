  <action>
Create Python script that:

1. **Imports and logging setup:**
   - json, os, subprocess, sys, time, hashlib
   - Structured JSON logging function: log_json(event, **kwargs)

2. **download_artifacts() function:**
   - Run: libcrs download-build-output coverage-harness /runner/artifacts/coverage-harness
   - Run: libcrs download-build-output compile-commands /runner/artifacts/compile-commands
   - Run: libcrs download-build-output callgraph /runner/artifacts/callgraph
   - Verify all paths exist, fail fast if missing

3. **register_seed_dirs() function:**
   - Run: libcrs register-fetch-dir /runner/seeds-in
   - Run: libcrs register-submit-dir seed /runner/seeds-out
   - Log registration success

4. **start_seedd(shared_dir) function:**
   - Start subprocess: ["/usr/local/bin/seedd", "--shared-dir", shared_dir]
   - Redirect stdout/stderr to PIPE
   - Health check loop (30 attempts, 1s sleep) using grpc health check
   - Return Popen object on success, raise RuntimeError on timeout
   - Log SeedD startup status

5. **main() function:**
   - Get env vars: TARGET_HARNESS, NUM_SEEDS (default 100), OSS_CRS_LLM_API_URL, OSS_CRS_LLM_API_KEY
   - Call download_artifacts()
   - Call register_seed_dirs()
   - Call start_seedd("/runner/shared")
   - Log "runner_ready" event
   - Placeholder loop: while True: time.sleep(60) with "waiting_for_seedgen" log
   - (Plan 02 will add actual SeedGenAgent calls)

6. **if __name__ == "__main__": main()**

Use error handling with log_json for all failures. Match patterns from 03-RESEARCH.md.
